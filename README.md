# Real_Estate
 AI-Powered Real Estate Intelligence Platform
📌 Project Overview
This project is an Artificial Intelligence and Machine Learning (AIML) based Real Estate Intelligence System that predicts property prices, analyzes investment risk, and provides smart investment recommendations (Buy / Hold / Sell).
It also includes future price forecasting and explainable AI insights to help users understand predictions.

🎯 Problem Statement
Real estate decision-making is often difficult due to:


Lack of accurate price prediction


Unclear investment risk


No structured decision support system


No future price forecasting tools


This project solves these problems using Machine Learning.

🚀 Features


🏠 Property Price Prediction using ML models


⚠️ Risk Analysis (Low / Medium / High)


📊 Investment Recommendation (Buy / Hold / Sell)


📈 Future Price Forecasting (1, 3, 5 years)


🧠 Explainable AI insights (feature importance idea)


📊 Data Analysis and Visualization



🧠 Machine Learning Models Used


Linear Regression


Random Forest Regressor


XGBoost Regressor


LightGBM Regressor


CatBoost Regressor


Gradient Boosting Regressor


Best Model Used: CatBoost / Ensemble Model

📊 Dataset


Contains ~14,000+ property records


Features include:


Bedrooms, Bathrooms


Living Area, Lot Area


Location (Latitude, Longitude, Postal Code)


Property Condition & Grade


Built Year, Renovation Year


Nearby Facilities
